# Complex Analysis — Interactive Tutorial

A self-contained, browser-based tutorial covering a full undergraduate Complex Analysis course (Brown & Churchill, 9th ed.). 23 modules from complex numbers up to evaluating real integrals via residues.

**Live site:** open `index.html` in any browser, or visit the GitHub Pages link (after enabling).

## Contents

1. Complex Numbers
2. Polar Form, Euler's Formula, Roots
3. Topology of ℂ
4. Limits & Continuity
5. Complex Differentiation
6. Cauchy–Riemann Equations
7. Analytic & Harmonic Functions
8. Möbius Transformations
9. Exponential & Logarithm
10. Powers & Trig Functions
11. Contour Integration
12. Cauchy–Goursat Theorem
13. Cauchy Integral Formula
14. Liouville & FTA
15. Maximum Modulus
16. Power Series
17. Taylor & Laurent Series
18. Singularities
19. Residues
20. Real Integrals (Rational)
21. Real Integrals (Trig + Jordan's Lemma)
22. Indented Paths & Branch Cuts
23. Final Summary

## Features

- MathJax-rendered formulas
- Click-to-reveal proofs and worked examples
- "Try it" boxes with hidden solutions
- Sidebar navigation, progress bar
- Single self-contained HTML file (no build, no dependencies beyond a CDN MathJax)

## Run locally

```
open index.html
```

That's it.
