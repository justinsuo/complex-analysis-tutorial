Module 1 — Complex Numbers

Why do we need complex numbers? To solve $x^2 + 1 = 0$. We invent $i$ with $i^2 = -1$, and suddenly every polynomial has roots.

1.1 Definition

A complex number is $z = x + iy$ where $x, y \in \mathbb{R}$ and $i^2 = -1$.
  • $\operatorname{Re}(z) = x$ — the real part
  • $\operatorname{Im}(z) = y$ — the imaginary part (just $y$, NOT $iy$ — this trips people up!)
The set of all complex numbers is $\mathbb{C}$.
You can identify $\mathbb{C}$ with $\mathbb{R}^2$ via $z = x+iy \leftrightarrow (x,y)$. They share the same metric and topology. But $\mathbb{C}$ has extra structure: a multiplication rule that $\mathbb{R}^2$ alone doesn't have.

1.2 Arithmetic

Addition is component-wise; multiplication uses $i^2 = -1$:

$$(x_1 + iy_1) + (x_2 + iy_2) = (x_1 + x_2) + i(y_1 + y_2)$$ $$(x_1 + iy_1)(x_2 + iy_2) = (x_1 x_2 - y_1 y_2) + i(x_1 y_2 + x_2 y_1)$$
$(2 + 3i)(1 - i) = 2 - 2i + 3i - 3i^2 = 2 + i + 3 = 5 + i$.

1.3 The trickiest formula: $1/z$

For $z = x + iy \neq 0$:

$$\frac{1}{z} = \frac{x}{x^2 + y^2} - i\,\frac{y}{x^2 + y^2}$$
Show derivation (it's a great trick: multiply by conjugate)
$\dfrac{1}{x+iy} = \dfrac{1}{x+iy} \cdot \dfrac{x-iy}{x-iy} = \dfrac{x-iy}{x^2+y^2}$. This trick — multiplying by the conjugate — is how you'll always simplify division.

1.4 Modulus and Conjugate

  • Modulus: $|z| = \sqrt{x^2 + y^2}$ — the length of the vector $(x,y)$.
  • Conjugate: $\bar z = x - iy$ — reflect across the real axis.

The single most useful identity in this whole class:

$$z \bar z = |z|^2$$

Proof: $(x+iy)(x-iy) = x^2 - (iy)^2 = x^2 + y^2$. Use this everywhere.

1.5 Properties to remember

  • $\overline{z_1 + z_2} = \bar z_1 + \bar z_2$ (and same for −, ×, ÷)
  • $|z_1 z_2| = |z_1||z_2|$, $|z_1/z_2| = |z_1|/|z_2|$
  • Triangle inequality: $|z_1 + z_2| \leq |z_1| + |z_2|$
  • Reverse triangle: $\big||z_1| - |z_2|\big| \leq |z_1 - z_2|$
  • $\operatorname{Re}(z) = (z + \bar z)/2$, $\operatorname{Im}(z) = (z - \bar z)/(2i)$
  • $z$ is real ⟺ $z = \bar z$; purely imaginary ⟺ $z = -\bar z$
Find $\operatorname{Re}$ and $\operatorname{Im}$ of $\dfrac{4+i}{2-3i}$.
Solution
Multiply by conjugate of denominator: $$\frac{4+i}{2-3i} \cdot \frac{2+3i}{2+3i} = \frac{(4+i)(2+3i)}{4+9} = \frac{8 + 12i + 2i - 3}{13} = \frac{5 + 14i}{13}.$$ So $\operatorname{Re} = 5/13$, $\operatorname{Im} = 14/13$.
Compute $(1+i)(1-i)$ and $|3 + 4i|$.
Answers
$(1+i)(1-i) = 1 - i^2 = 2$. And $|3+4i| = \sqrt{9+16} = 5$.

Module 2 — Polar Form, Euler's Formula, Roots

Cartesian form $x+iy$ is great for adding. Polar form $re^{i\theta}$ is great for multiplying. We need both.

2.1 Polar form

Any nonzero complex number can be written as

$$z = r(\cos\theta + i \sin\theta)$$

where $r = |z|$ and $\theta$ is the angle from the positive real axis.

2.2 Euler's formula

For any real $\theta$: $\quad e^{i\theta} = \cos\theta + i\sin\theta$.

Why is this defined this way? Because we want $e^{iz}$ to satisfy its Taylor series. If you plug $iy$ into $e^z = \sum z^n/n!$, separate into real/imaginary parts, you get exactly $\cos y + i\sin y$. So this is the only consistent definition.

$$z = re^{i\theta}, \quad \bar z = re^{-i\theta}$$

2.3 The killer feature: multiplication

$$z_1 z_2 = r_1 r_2 \, e^{i(\theta_1 + \theta_2)}$$ Multiply moduli, add angles.
$(1+i)(1+i)$ in polar: $1+i = \sqrt 2 \, e^{i\pi/4}$, so square it gives $2 \, e^{i\pi/2} = 2i$. ✓ (matches $(1+i)^2 = 2i$).

2.4 De Moivre's formula

By induction:

$$z^n = r^n e^{in\theta} \quad \text{for any integer } n$$
Compute $(-1+i)^7$.
Solution
$-1+i = \sqrt 2\, e^{i 3\pi/4}$. So $(-1+i)^7 = (\sqrt 2)^7 e^{i 21\pi/4}$. Reduce $21\pi/4$ mod $2\pi$: $21\pi/4 - 4\pi = 5\pi/4$. So $= 8\sqrt 2 \cdot e^{i 5\pi/4} = 8\sqrt 2 \,(-1/\sqrt 2 - i/\sqrt 2) = -8 - 8i$.

2.5 Argument is multivalued!

$\theta$ is not unique: you can add $2\pi n$ for any integer $n$ and get the same $z$. We write $$\arg z = \{\theta + 2n\pi : n \in \mathbb{Z}\}$$ This multivaluedness comes back to bite us with $\log z$ and $z^c$.
The principal argument $\operatorname{Arg} z$ is the unique value in $(-\pi, \pi]$.

2.6 $k$-th roots

The equation $w^k = z$ where $z = re^{i\theta} \neq 0$ has exactly $k$ solutions: $$w_n = r^{1/k}\, e^{i(\theta + 2n\pi)/k}, \quad n = 0, 1, \ldots, k-1.$$
Find $\sqrt[4]{-16}$.
Solution
$-16 = 16 e^{i\pi}$. So $(-16)^{1/4} = 2 \cdot e^{i(\pi + 2n\pi)/4}$ for $n=0,1,2,3$.
  • $n=0$: $2e^{i\pi/4} = \sqrt 2 + i\sqrt 2$
  • $n=1$: $2e^{i 3\pi/4} = -\sqrt 2 + i\sqrt 2$
  • $n=2$: $2e^{i 5\pi/4} = -\sqrt 2 - i\sqrt 2$
  • $n=3$: $2e^{i 7\pi/4} = \sqrt 2 - i\sqrt 2$
The 4 roots sit at the corners of a square centered at 0.
Find all values of $\sqrt[3]{1}$ (cube roots of unity).
Answer
$1, e^{2\pi i/3}, e^{4\pi i/3}$, equivalently $1, \tfrac{-1+i\sqrt 3}{2}, \tfrac{-1 - i\sqrt 3}{2}$.

Module 3 — Topology of $\mathbb{C}$

Most of this is identical to $\mathbb{R}^2$ — same distance, same open/closed sets. But we need new vocabulary: simply connected matters a lot later.

3.1 Distance and neighborhoods

The $r$-neighborhood of $z_0$ is $B_r(z_0) = \{z : |z - z_0| < r\}$ — an open disk.
The deleted $r$-neighborhood is $0 < |z - z_0| < r$ — disk minus center.

3.2 Open / closed / boundary

A set $S \subseteq \mathbb{C}$ is open if every point of $S$ has some $r$-neighborhood entirely inside $S$.
$S$ is closed if its complement is open (equivalently, $S$ contains all its limit points).
The boundary $\partial S$ = points where every neighborhood meets both $S$ and its complement.

3.3 Connected and simply connected

$S$ is (path-)connected if any two points in $S$ can be joined by a continuous path lying inside $S$.
A domain is an open + connected subset of $\mathbb{C}$. We use $D$ for domains throughout.
$D$ is simply connected if every closed curve in $D$ can be continuously shrunk to a point within $D$.
Equivalent (and more practical!) condition in $\mathbb{C}$:
$D$ simply connected ⟺ for every simple closed curve $C \subset D$, the interior of $C$ is also in $D$.
In plain English: no holes.

Examples

  • $\mathbb{C}$ itself ✓ simply connected
  • An open disk ✓ simply connected
  • $\mathbb{C} \setminus \{0\}$ ✗ not simply connected — has a hole at 0
  • $\{re^{i\theta}: r>0,\ -\pi < \theta < \pi\}$ (slit plane) ✓ simply connected — we removed an entire ray, no hole left
Why care? Many theorems (Cauchy–Goursat, antiderivative existence, branches of log) require simply connected domains. The whole reason $\log z$ needs a branch cut is that $\mathbb{C} \setminus \{0\}$ isn't simply connected.
Is $\{z : 1 < |z| < 2\}$ (an annulus) simply connected?
Answer
No — it has a hole (the disk $|z|\leq 1$ is missing). A loop around 0 can't be shrunk to a point inside the annulus.

Module 4 — Limits and Continuity

For limits and continuity, complex analysis is basically the same as real $\mathbb{R}^2 \to \mathbb{R}^2$. The big difference comes with differentiation next module.

4.1 Complex functions

A complex function $f: \mathbb{C} \to \mathbb{C}$ can always be written

$$f(z) = u(x,y) + i\, v(x,y)$$

where $u, v: \mathbb{R}^2 \to \mathbb{R}$ are the real and imaginary parts. There's a 1-1 correspondence between $f$ and the pair $(u,v)$.

$f(z) = z^2$. With $z = x+iy$, expand: $z^2 = (x^2-y^2) + 2xyi$. So $u = x^2 - y^2$, $v = 2xy$.

4.2 Limits

$\displaystyle\lim_{z \to z_0} f(z) = w_0$ means: for every $\varepsilon > 0$ there exists $\delta > 0$ such that $$0 < |z - z_0| < \delta \implies |f(z) - w_0| < \varepsilon.$$
With $f = u + iv$, $z_0 = x_0 + iy_0$, $w_0 = u_0 + iv_0$: $$\lim_{z\to z_0} f = w_0 \iff \lim u = u_0 \text{ and } \lim v = v_0.$$

4.3 Limits at and to infinity

There's only one $\infty$ in $\mathbb{C}$ (no $\pm\infty$). The trick: use $1/z$.

  • $\lim_{z\to z_0} f(z) = \infty \iff \lim_{z\to z_0} 1/f(z) = 0$
  • $\lim_{z\to\infty} f(z) = w_0 \iff \lim_{z\to 0} f(1/z) = w_0$
$\displaystyle\lim_{z \to \infty} \frac{2z + i}{z+1}$. Apply the trick: replace $z$ with $1/z$ and let $z\to 0$: $$\frac{2/z + i}{1/z + 1} = \frac{2 + iz}{1 + z} \xrightarrow{z\to 0} 2.$$

4.4 Continuity

$f$ is continuous at $z_0$ if $\lim_{z\to z_0} f(z) = f(z_0)$.
$f$ continuous ⟺ $u$ and $v$ both continuous (as real functions of $(x,y)$).
All the usual rules apply: sums, products, quotients (where denom ≠ 0), and compositions of continuous functions are continuous.

Module 5 — Complex Differentiation

Here is where everything diverges from $\mathbb{R}^2$. The complex derivative is much stronger than partial derivatives — and that strength forces the Cauchy–Riemann equations.

5.1 The definition (looks innocent)

$f$ is (complex) differentiable at $z_0$ if $$f'(z_0) = \lim_{\Delta z \to 0} \frac{f(z_0 + \Delta z) - f(z_0)}{\Delta z}$$ exists.
The catch: $\Delta z$ can approach 0 from any direction in the complex plane. The limit must exist and be the same regardless of how $\Delta z \to 0$. This is a much stronger condition than in 1D real calculus.

5.2 First example: $f(z) = z^2$

$$\frac{(z+\Delta z)^2 - z^2}{\Delta z} = 2z + \Delta z \to 2z.$$ So $(z^2)' = 2z$. Same as real case.

5.3 The shocking example: $f(z) = \bar z$

Let's try the limit:

$$\frac{\overline{z + \Delta z} - \bar z}{\Delta z} = \frac{\overline{\Delta z}}{\Delta z}$$

Now compute this limit two ways:

  • $\Delta z = \Delta x$ (real direction): ratio $= \Delta x / \Delta x = 1$.
  • $\Delta z = i\Delta y$ (imaginary direction): ratio $= -i\Delta y / (i\Delta y) = -1$.

Different limits! So $f(z) = \bar z$ is nowhere differentiable, even though it's smooth as a function of $(x,y)$.

This is the moral: complex differentiability is a strong, restrictive condition.

5.4 Standard rules (same as real)

  • $(f+g)' = f' + g'$
  • $(fg)' = f'g + fg'$
  • $(f/g)' = (f'g - fg')/g^2$
  • Chain rule: $(g \circ f)'(z) = g'(f(z)) \cdot f'(z)$
  • $(z^n)' = n z^{n-1}$ for any integer $n$

5.5 Notation

$f'(z)$, $\dfrac{df}{dz}$, $\dfrac{d}{dz}f$ all mean the same thing.

Module 6 — Cauchy–Riemann Equations ⭐ KEY

The CR equations are the engine of complex analysis. They tell you exactly when a function is complex-differentiable in terms of partial derivatives of $u$ and $v$.

6.1 The big theorem

Let $f(z) = u(x,y) + iv(x,y)$. Then $f$ is complex-differentiable at $z_0$ if and only if:
  1. $u, v$ are real-differentiable at $(x_0, y_0)$, AND
  2. The Cauchy–Riemann equations hold: $$u_x = v_y, \qquad u_y = -v_x.$$
And in that case: $$f'(z_0) = u_x + iv_x = v_y - iu_y.$$

6.2 Where do the CR equations come from?

Show the slick derivation
Take $\Delta z \to 0$ from two directions:
  • $\Delta z = \Delta x$ (along x-axis): limit becomes $\dfrac{\partial f}{\partial x} = u_x + iv_x$.
  • $\Delta z = i\Delta y$ (along y-axis): limit becomes $\dfrac{1}{i}\dfrac{\partial f}{\partial y} = v_y - iu_y$ (since $1/i = -i$).
For the limit to exist, both must agree. Equating real and imaginary parts: $$u_x = v_y, \quad v_x = -u_y. \quad \blacksquare$$

6.3 A practical sufficient condition

If $u_x, u_y, v_x, v_y$ are continuous in a neighborhood and CR holds — then $f$ is differentiable.
In practice, partials are almost always continuous, so we just check CR.

6.4 Worked example

Is $f(z) = z^2$ differentiable?
$u = x^2 - y^2$, $v = 2xy$. Compute partials:
  • $u_x = 2x$, $u_y = -2y$
  • $v_x = 2y$, $v_y = 2x$
Check CR: $u_x = 2x = v_y$ ✓; $u_y = -2y = -v_x$ ✓.
All continuous, CR holds everywhere ⟹ $f$ entire. And $f' = u_x + iv_x = 2x + 2yi = 2z$. ✓

6.5 Classic non-example: $f(z) = \bar z$

$u = x$, $v = -y$. So $u_x = 1$, $v_y = -1$. Already $u_x \neq v_y$, so CR fails. $\bar z$ is nowhere differentiable.

6.6 CR in polar coordinates

If you write $f$ using $r$ and $\theta$:

$f$ differentiable ⟺ $u, v$ real-differentiable AND $$r u_r = v_\theta, \qquad u_\theta = -r v_r.$$ And $f'(z) = e^{-i\theta}(u_r + i v_r)$.

Useful when $f$ is naturally expressed in polar form (e.g. $\sqrt z$, $\log z$).

6.7 The slick formulation: $\partial/\partial \bar z$

Define

$$\frac{\partial}{\partial z} = \frac{1}{2}\!\left(\frac{\partial}{\partial x} - i\frac{\partial}{\partial y}\right), \quad \frac{\partial}{\partial \bar z} = \frac{1}{2}\!\left(\frac{\partial}{\partial x} + i\frac{\partial}{\partial y}\right)$$
$f$ satisfies CR ⟺ $\dfrac{\partial f}{\partial \bar z} = 0$.
And if so, $f' = \dfrac{\partial f}{\partial z}$.
Slogan: "$f$ is analytic ⟺ $f$ has no $\bar z$ in it."
Quick test: $|z|^2 = z\bar z$ has a $\bar z$, so it's not analytic (except where $\partial/\partial \bar z = z = 0$).
For $f(z) = e^z = e^x(\cos y + i\sin y)$: write $u, v$ and verify CR.
Solution
$u = e^x \cos y$, $v = e^x \sin y$. $u_x = e^x \cos y$, $v_y = e^x \cos y$. ✓
$u_y = -e^x \sin y$, $v_x = e^x \sin y$, so $u_y = -v_x$. ✓
Hence $f$ is entire and $f'(z) = u_x + iv_x = e^x \cos y + i e^x \sin y = e^z$. (Same as real exponential!)

Module 7 — Analytic & Harmonic Functions

"Analytic" means differentiable in an open neighborhood, not just at one point. This subtle distinction unlocks all the deep theorems.

7.1 Definition

$f$ is analytic at $z_0$ if it is differentiable on some open disk around $z_0$.
$f$ is analytic on a domain $D$ if it's differentiable at every point of $D$.
$f$ is entire if it's analytic on all of $\mathbb{C}$.

7.2 Why "open neighborhood" matters

$f(z) = |z|^2 = x^2 + y^2$. CR gives $2x = 0, 2y = 0$, only at $z=0$. So $f$ is differentiable only at $z=0$, but not analytic anywhere — there's no open disk where $f$ is differentiable.

7.3 Constancy theorem

If $f$ is analytic on a domain $D$ and $f'(z) = 0$ everywhere on $D$, then $f$ is constant.
Idea of proof
$f' = 0$ ⟹ $u_x = u_y = v_x = v_y = 0$ ⟹ $u, v$ constant on the connected $D$ ⟹ $f$ constant.

Friend corollaries

  • If $f$ and $\bar f$ are both analytic on $D$ ⟹ $f$ is constant.
  • If $|f|$ is constant on $D$ ⟹ $f$ is constant.
  • If $f$ takes only real values on $D$ ⟹ $f$ is constant.

7.4 Harmonic functions

A real-valued function $u(x,y)$ is harmonic if $u_{xx} + u_{yy} = 0$ (Laplace equation), with continuous second partials.
If $f = u + iv$ is analytic on $D$, then both $u$ and $v$ are harmonic on $D$.
Proof
From CR: $u_x = v_y$, so $u_{xx} = v_{yx}$. Also $u_y = -v_x$, so $u_{yy} = -v_{xy}$. Adding: $u_{xx} + u_{yy} = v_{yx} - v_{xy} = 0$ (mixed partials equal when continuous). Same for $v$.

7.5 Harmonic conjugate

If $u$ is harmonic on a simply connected domain, there exists $v$ (a "harmonic conjugate") such that $f = u+iv$ is analytic.
Simply connected is essential! Counterexample: $u = \ln\sqrt{x^2+y^2}$ is harmonic on $\mathbb{C}\setminus\{0\}$ — but no global $v$ exists. (The natural choice would be $v = \theta$, but $\theta$ isn't single-valued on $\mathbb{C}\setminus\{0\}$.)

Module 8 — Möbius Transformations

A small but elegant detour. These maps connect analysis, geometry, and algebra in a beautiful way.

8.1 Definition

A Möbius transformation is $$f(z) = \frac{az + b}{cz + d}, \quad ad - bc \neq 0.$$

The condition $ad - bc \neq 0$ ensures $f$ isn't constant.

8.2 Properties

  • $f'(z) = \dfrac{ad - bc}{(cz+d)^2}$
  • $f$ is invertible: $f^{-1}(w) = \dfrac{-dw + b}{cw - a}$
  • The inverse is also Möbius — so Möbius transformations form a group under composition.

8.3 Generators

Every Möbius transformation is a composition of:
  1. Translation: $z \mapsto z + b$
  2. Dilation: $z \mapsto rz$ (real $r > 0$)
  3. Rotation: $z \mapsto e^{i\theta}z$
  4. Inversion: $z \mapsto 1/z$

8.4 The geometric magic: circles → circles

Every Möbius transformation maps "circles" to "circles", where a line counts as a circle through $\infty$.

The proof reduces to inversion: any circle in the plane has equation $a(x^2+y^2) + bx + cy = d$, and a calculation shows $1/z$ maps such an equation to another of the same form.

$1/z$ maps the line $x = 1$ to a circle: substituting $z = 1+iy$, $1/z = (1-iy)/(1+y^2)$, so $u = 1/(1+y^2)$, $v = -y/(1+y^2)$. You can verify $u^2 + v^2 = u$, i.e., $(u - 1/2)^2 + v^2 = 1/4$ — a circle of radius $1/2$.

Module 9 — Exponential and Logarithm

$e^z$ is the simplest analytic function. $\log z$ is the trickiest — and a great window into multivalued functions and branch cuts.

9.1 Complex exponential

$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

Key facts

  • $e^z$ is entire, with $(e^z)' = e^z$.
  • $e^{z_1 + z_2} = e^{z_1} e^{z_2}$ ✓
  • $|e^z| = e^x$ (so $|e^z| \neq 0$ ever, but can be huge or tiny).
  • Periodic with period $2\pi i$: $e^{z_1} = e^{z_2} \iff z_1 - z_2 = 2n\pi i$. (This is HUGE.)
  • Range of $e^z$ = $\mathbb{C} \setminus \{0\}$. Yes, you can have $e^z = -1$: $z = i\pi$ does it.

9.2 Complex logarithm

We want $\log z$ to satisfy $e^{\log z} = z$. Since $e^z$ is periodic, taking $\log$ is multivalued:

For $z = re^{i\theta} \neq 0$: $$\log z = \ln r + i(\theta + 2n\pi), \quad n \in \mathbb{Z}.$$

Real part is uniquely $\ln |z|$, but imaginary part has infinitely many values differing by $2\pi i$.

$\log(1) = 0, 2\pi i, 4\pi i, \ldots$ ✓ (since $e^{2\pi i k} = 1$).
$\log(-1) = i\pi + 2n\pi i$.
$\log(i) = i\pi/2 + 2n\pi i$.

9.3 Principal log

The principal log $\operatorname{Log} z = \ln r + i\Theta$, where $\Theta = \operatorname{Arg} z \in (-\pi, \pi]$.
$\operatorname{Log} z$ is not continuous on the negative real axis! Approaching $-1$ from above: $\theta \to \pi$. From below: $\theta \to -\pi$. Two different limits ⟹ discontinuity.

9.4 Branches

A branch of $\log z$ on a domain $D$ is a single-valued analytic function on $D$ whose value is one of the values of $\log z$.
On the slit plane $D = \{re^{i\theta}: r > 0,\ -\pi < \theta < \pi\}$, the principal log is analytic with $$(\operatorname{Log} z)' = \frac{1}{z}.$$

The branch cut is the negative real axis (the set we removed). Cross it and the function jumps.

General branches

Pick any $\alpha \in \mathbb{R}$. On $\{r > 0, \alpha < \theta < \alpha + 2\pi\}$, define $f(z) = \ln r + i\theta$. This is analytic — the branch cut is the ray $\theta = \alpha$.

9.5 Why no antiderivative for $1/z$ on $\mathbb{C}\setminus\{0\}$

We'll see later that $\oint_{|z|=1} dz/z = 2\pi i \neq 0$, which (via a theorem we'll prove) means $1/z$ has no antiderivative on $\mathbb{C}\setminus\{0\}$. But on the slit plane (simply connected), $\operatorname{Log} z$ is an antiderivative.

Compute $\operatorname{Log}(-1 - \sqrt 3 i)$.
Solution
$|-1 - \sqrt 3 i| = 2$. $\operatorname{Arg} = -2\pi/3$ (third quadrant, principal value in $(-\pi, \pi]$).
So $\operatorname{Log}(-1-\sqrt 3 i) = \ln 2 - 2\pi i/3$.

Module 10 — Powers, Exponentials with Other Bases, Trig

10.1 General power $z^c$

For $z \neq 0$ and any complex $c$: $$z^c = e^{c \log z}.$$

How many values? It depends on $c$:

  • $c \in \mathbb{Z}$: 1 value (no branch needed)
  • $c = p/q$ in lowest terms: $q$ values
  • $c$ irrational or non-real: infinitely many values

Branch of $z^c$

Same idea as log: pick a branch of $\log z$ (cut a ray), and $z^c = e^{c \cdot (\text{branch of } \log)}$ is single-valued and analytic on the slit plane, with

$$(z^c)' = c\, z^{c-1}.$$
Compute the principal value of $i^i$.
Solution
$i = e^{i\pi/2}$, so $\operatorname{Log} i = i\pi/2$. Then $$i^i = e^{i \cdot i\pi/2} = e^{-\pi/2}.$$ Yes, that's a real number! (All values: $e^{-\pi/2 - 2n\pi}$.)

10.2 Exponential with base $c$

For $c \neq 0$: $$c^z = e^{z \log c}.$$
Here $z$ is in the exponent, not in the log. So once you fix one branch of $\log c$ (i.e., pick one value), $c^z$ is entire! No branch cut needed.

10.3 Trigonometric functions

Inspired by $e^{ix} = \cos x + i\sin x$ and $e^{-ix} = \cos x - i\sin x$:

$$\sin z = \frac{e^{iz} - e^{-iz}}{2i}, \qquad \cos z = \frac{e^{iz} + e^{-iz}}{2}.$$

Properties (mostly the same as real)

  • $(\sin z)' = \cos z$, $(\cos z)' = -\sin z$ — entire functions.
  • $\sin^2 z + \cos^2 z = 1$ ✓
  • All addition formulas hold: $\sin(z_1+z_2) = \sin z_1 \cos z_2 + \cos z_1 \sin z_2$, etc.
  • $\sin(-z) = -\sin z$ (odd), $\cos(-z) = \cos z$ (even).

The shocker: not bounded!

$\sin(iy) = (e^{-y} - e^y)/(2i) = i\sinh y$. As $y \to \infty$, $|\sin(iy)| \to \infty$. So $\sin, \cos$ are unbounded on $\mathbb{C}$.

Zeros

$\sin z = 0 \iff z = n\pi$, $\cos z = 0 \iff z = (n + \tfrac{1}{2})\pi$. (All real!)

10.4 Hyperbolic functions

$$\sinh z = \frac{e^z - e^{-z}}{2}, \qquad \cosh z = \frac{e^z + e^{-z}}{2}.$$

Useful identities:

  • $\sinh(iz) = i\sin z$, $\cosh(iz) = \cos z$ — sinh/cosh and sin/cos are essentially the same in complex world!
  • $\cosh^2 z - \sinh^2 z = 1$.
  • $\sinh z = 0 \iff z = n\pi i$, $\cosh z = 0 \iff z = (n+\tfrac{1}{2})\pi i$. (All purely imaginary!)

10.5 Inverse trig

Solve $\sin w = z$ by setting $t = e^{iw}$ → quadratic equation in $t$:

$$\sin^{-1} z = -i\, \log\!\left(iz + (1-z^2)^{1/2}\right)$$ $$\cos^{-1} z = -i\, \log\!\left(z + i(1-z^2)^{1/2}\right)$$

Both are multivalued; pick branches. Derivatives match real case: $(\sin^{-1} z)' = 1/\sqrt{1-z^2}$.

Compute all values of $\sin^{-1} i$.
Solution
$\sin^{-1} i = -i\log(i\cdot i + (1 - i^2)^{1/2}) = -i \log(-1 + \sqrt 2)$.
$-1 + \sqrt 2 > 0$, so $\log(-1 + \sqrt 2) = \ln(\sqrt 2 - 1) + 2n\pi i$.
Or take the other root $\sqrt{}$ value: $-1 - \sqrt 2 < 0$, so $\log = \ln(\sqrt 2 + 1) + i\pi + 2n\pi i$.
Multiplying by $-i$ gives the values.

Module 11 — Contour Integration

In real calculus you integrate over intervals. In complex analysis, you integrate over curves (contours). Everything that follows depends on this.

11.1 Integrals of complex-valued functions of a real variable

For $w(t) = u(t) + iv(t)$ continuous on $[a,b]$:

$$\int_a^b w(t)\,dt = \int_a^b u(t)\,dt + i \int_a^b v(t)\,dt.$$

Real and imaginary parts integrate separately. FTC: $\int_a^b W'(t)\,dt = W(b) - W(a)$.

11.2 Contours

A contour $C$ is a piecewise-smooth curve, parametrized by $z(t): [a,b] \to \mathbb{C}$ with $z'(t) \neq 0$ except at finitely many corners.

11.3 Contour integral

$$\int_C f(z)\,dz = \int_a^b f(z(t))\, z'(t)\, dt.$$

Recipe

  1. Parametrize $C$ as $z(t)$.
  2. Substitute $z = z(t)$ into $f$.
  3. Multiply by $z'(t)$.
  4. Integrate the resulting (complex-valued) function of $t$ over $[a,b]$.
Compute $\int_C z^2\, dz$ where $C$ is the upper semicircle $|z|=1$, from $1$ to $-1$.
Solution
Parametrize: $z(\theta) = e^{i\theta}$, $\theta \in [0, \pi]$. Then $z' = ie^{i\theta}$, $z^2 = e^{2i\theta}$. $$\int_0^\pi e^{2i\theta}\cdot ie^{i\theta}\,d\theta = i\int_0^\pi e^{3i\theta}\,d\theta = i \cdot \left[\frac{e^{3i\theta}}{3i}\right]_0^\pi = \frac{e^{3i\pi}-1}{3} = \frac{-1-1}{3} = -\frac{2}{3}.$$

11.4 Length of a contour

$$L = \int_a^b |z'(t)|\, dt$$

11.5 Crucial inequality (the "ML inequality")

If $|f(z)| \leq M$ on $C$ and $C$ has length $L$, then $$\left| \int_C f(z)\, dz \right| \leq M L.$$
We use this everywhere to show integrals vanish.

11.6 Antiderivative theorem

For $f$ continuous on a domain $D$, the following are equivalent:
  1. (A) $f$ has an antiderivative $F$ on $D$ (i.e., $F'=f$).
  2. (B) $\int_C f\,dz$ depends only on the endpoints of $C$, not the path.
  3. (C) $\int_C f\,dz = 0$ for every closed contour $C$ in $D$.
And if (A) holds, $\int_{z_1}^{z_2} f\,dz = F(z_2) - F(z_1)$.

Big example: $f(z) = 1/z$ on $\mathbb{C}\setminus\{0\}$

$\oint_{|z|=1} \dfrac{dz}{z} = \int_0^{2\pi} \dfrac{1}{e^{i\theta}}\, ie^{i\theta}\,d\theta = \int_0^{2\pi} i\,d\theta = 2\pi i \neq 0$.
So $1/z$ has no antiderivative on $\mathbb{C}\setminus\{0\}$.
But on the slit plane (simply connected), $\operatorname{Log} z$ is an antiderivative.

Module 12 — Cauchy–Goursat Theorem ⭐⭐

If a function is analytic, integrals around closed loops vanish. This is the cornerstone of everything that follows.

12.1 The theorem

Let $C$ be a simple closed contour and $f$ analytic on and inside $C$. Then $$\oint_C f(z)\, dz = 0.$$
Show easy proof using Green's theorem (assumes $f'$ continuous)
Write $f = u + iv$, $dz = dx + i\,dy$. So $$\oint_C f\,dz = \oint_C (u\,dx - v\,dy) + i\oint_C (v\,dx + u\,dy).$$ By Green's theorem (need continuous partials): $$= \iint_R (-v_x - u_y)\,dA + i \iint_R (u_x - v_y)\,dA.$$ By CR: $u_x = v_y$ and $u_y = -v_x$, so both integrands are $0$. Done.
Goursat's full proof (no extra assumption — sketch)
Divide the region into many small squares. On each tiny square, write $f(z) = f(z_j) + f'(z_j)(z - z_j) + \delta_j(z)(z - z_j)$ with $\delta_j \to 0$. The constant and linear parts have antiderivatives, so their loop integrals are 0. The remainder is shown to vanish via the ML inequality as squares shrink. Adding contributions, interior edges cancel and only the original $C$ remains, giving 0.

12.2 Massive corollary: simply connected case

If $D$ is simply connected and $f$ is analytic on $D$, then $\oint_C f\,dz = 0$ for every closed $C$ in $D$. So $f$ has an antiderivative on $D$.

12.3 Multiply connected domains: deformation

If $C_1, C_2$ are simple closed contours, $C_1$ inside $C_2$, both counterclockwise, and $f$ analytic in the region between them, then $$\oint_{C_1} f\,dz = \oint_{C_2} f\,dz.$$

Picture: cut the annular region with two crosscuts to make a simply connected region whose boundary is $C_2$ minus $C_1$.

Compute $\oint_C \dfrac{dz}{z}$ for any simple closed $C$ enclosing 0.
Solution
By the deformation theorem, this equals the integral over the unit circle, which we computed: $2\pi i$. So any loop enclosing 0 gives $2\pi i$. (And any loop NOT enclosing 0 gives 0 by Cauchy–Goursat.)

12.4 More general version

If $C$ is a simple closed (counterclockwise) contour, and $C_1, \ldots, C_n$ are simple closed contours inside $C$ (each counterclockwise, mutually disjoint), and $f$ analytic in the region inside $C$ but outside $C_1, \ldots, C_n$: $$\oint_C f\,dz = \sum_{k=1}^n \oint_{C_k} f\,dz.$$

Used when $f$ has multiple isolated singularities — wrap a small loop around each.

Module 13 — Cauchy Integral Formula ⭐⭐⭐

This is the magical formula. It says: if $f$ is analytic, the value inside the contour is determined by values on the contour. From this, all derivatives exist!

13.1 The CIF

Let $C$ be a simple closed contour (counterclockwise), $f$ analytic on and inside $C$, and $z_0$ inside $C$. Then $$f(z_0) = \frac{1}{2\pi i}\oint_C \frac{f(z)}{z - z_0}\, dz.$$
Proof sketch
$f(z)/(z-z_0)$ is analytic on the region bounded by $C$ and a small circle $C_\rho$ of radius $\rho$ around $z_0$. By the deformation theorem, $$\oint_C \frac{f(z)}{z-z_0}\,dz = \oint_{C_\rho} \frac{f(z)}{z-z_0}\,dz.$$ On $C_\rho$, parametrize $z = z_0 + \rho e^{i\theta}$. As $\rho \to 0$, $f(z) \to f(z_0)$ (continuity), and the integral approaches $f(z_0) \int_0^{2\pi} i\,d\theta = 2\pi i\, f(z_0)$.

13.2 Extension: all derivatives

Under the same hypotheses: $$f^{(n)}(z_0) = \frac{n!}{2\pi i}\oint_C \frac{f(z)}{(z - z_0)^{n+1}}\,dz.$$

13.3 Earth-shattering consequences

Analytic ⟹ infinitely differentiable.

Why: the CIF expresses $f(z_0)$ as an integral. Differentiate under the integral sign — you can differentiate $1/(z-z_0)$ as many times as you want. So all derivatives $f^{(n)}$ exist.

If $f$ has an antiderivative on $D$, then $f$ is analytic.

Why: if $F' = f$, then $F$ analytic ⟹ $F$ has all derivatives ⟹ $F''$ exists ⟹ $f' = F''$ exists ⟹ $f$ analytic.

13.4 Worked example

Compute $\oint_{|z|=1} \dfrac{\cos z}{z^2 + 9}\,dz$.
Solution
The denominator is zero at $z = \pm 3i$, both outside the unit circle. So the integrand factor $\cos z/(z^2+9)$ is analytic inside. Treat as $f(z)/z$ where $f(z) = \cos z/(z^2+9)$: $$\oint \frac{f(z)}{z - 0}\,dz = 2\pi i\, f(0) = 2\pi i \cdot \frac{1}{9} = \frac{2\pi i}{9}.$$
Compute $\oint_{|z|=1} \dfrac{e^{2z}}{z^4}\,dz$.
Solution
Use the extended formula with $f(z) = e^{2z}$, $z_0 = 0$, $n = 3$ (since $z^4 = z^{n+1}$): $$\oint \frac{e^{2z}}{z^4}\,dz = \frac{2\pi i}{3!}\, f^{(3)}(0).$$ $f^{(3)}(z) = 8e^{2z}$, so $f^{(3)}(0) = 8$. Answer: $\dfrac{2\pi i \cdot 8}{6} = \dfrac{8\pi i}{3}$.

Module 14 — Liouville & Fundamental Theorem of Algebra

14.1 Liouville's theorem

A bounded entire function is constant.
Proof (clean and short!)
Suppose $|f(z)| \leq M$ for all $z$. Pick any $z_0$ and $R > 0$. By the CIF derivative formula on the circle $|z - z_0| = R$: $$|f'(z_0)| = \left|\frac{1}{2\pi i}\oint \frac{f(z)}{(z-z_0)^2}\,dz\right| \leq \frac{1}{2\pi}\cdot 2\pi R \cdot \frac{M}{R^2} = \frac{M}{R}.$$ Let $R \to \infty$: $f'(z_0) = 0$. Since $z_0$ was arbitrary, $f' \equiv 0$, so $f$ is constant. $\blacksquare$
This dramatically fails in real analysis! $\sin x$ is bounded and infinitely differentiable on $\mathbb{R}$. The complex sine, by contrast, is unbounded — and that's required, by Liouville.

14.2 Fundamental Theorem of Algebra

Every non-constant complex polynomial has at least one root in $\mathbb{C}$.
Proof using Liouville
Suppose $P(z)$ has no roots. Then $1/P(z)$ is entire. We show it's bounded.
For $|z|$ large enough, the leading term $a_n z^n$ dominates and $|P(z)| \geq |a_n||z|^n / 2$, so $|1/P(z)|$ is small. On any bounded compact set, $1/P$ is continuous, hence bounded.
So $1/P$ is entire and bounded ⟹ constant ⟹ $P$ constant. Contradiction.

Stronger form

A degree-$n$ polynomial $P(z) = a_n z^n + \cdots + a_0$ factors as $$P(z) = a_n(z - z_1)(z - z_2)\cdots(z - z_n)$$ (the $z_k$ may repeat).

By induction: find one root $z_1$, divide out $(z - z_1)$, get a degree $(n-1)$ poly, repeat.

Module 15 — Maximum Modulus Principle

15.1 The principle

If $f$ is analytic on a domain $D$ and $|f|$ attains a maximum at some interior point $z_0 \in D$, then $f$ is constant on $D$.

In other words: $|f|$ can only attain its max on the boundary (unless $f$ is constant).

15.2 Local version

If $f$ is analytic on a disk $|z - z_0| < R$ and $|f(z)| \leq |f(z_0)|$ for all $z$ in this disk, then $f$ is constant on the disk.
Proof using CIF (mean value property)
For any $r < R$, $f(z_0) = \frac{1}{2\pi}\int_0^{2\pi} f(z_0 + re^{i\theta})\,d\theta$ (CIF on circle of radius $r$).
Take absolute values, apply $|f| \leq |f(z_0)|$ inside the integral: $$|f(z_0)| \leq \frac{1}{2\pi}\int_0^{2\pi} |f(z_0 + re^{i\theta})|\,d\theta \leq |f(z_0)|.$$ Both bounds match ⟹ $|f(z_0 + re^{i\theta})| = |f(z_0)|$ for all $\theta$. So $|f|$ is constant on each circle, hence on the disk. Then $f$ is constant.

The global version follows: cover any path from one point to another with overlapping disks.

15.3 Useful applications

  • If $f$ analytic on a closed bounded region, $|f|$ attains its max on the boundary.
  • Schwarz lemma (variant): if $f$ analytic on $|z|<1$, $f(0)=0$, $|f|\leq 1$, then $|f(z)| \leq |z|$ everywhere.

Module 16 — Sequences and Power Series

16.1 Sequences

$\{z_n\}$ converges to $z_0$ if $|z_n - z_0| \to 0$. Equivalent to: real and imaginary parts both converge.

$\sum z_n$ converges if partial sums converge. Standard tests work (absolute convergence, comparison, ratio).

16.2 Power series

A power series centered at $z_0$ is $$\sum_{n=0}^\infty a_n (z - z_0)^n.$$

16.3 Radius of convergence

For each power series, there exists $R \in [0, \infty]$ — the radius of convergence — such that:
  • The series converges absolutely for $|z - z_0| < R$.
  • It diverges for $|z - z_0| > R$.
  • On $|z - z_0| < R$, convergence is uniform on compact subsets.
The formula: $$\frac{1}{R} = \limsup_{n\to\infty} |a_n|^{1/n}.$$
$\sum z^n$: $a_n = 1$, $\limsup |a_n|^{1/n} = 1$, so $R = 1$. Converges for $|z| < 1$ to $1/(1-z)$.

16.4 Power series define analytic functions

A power series with radius $R > 0$ defines an analytic function on $|z - z_0| < R$, and you can differentiate term by term: $$f'(z) = \sum_{n=1}^\infty n\, a_n (z - z_0)^{n-1},$$ also valid on $|z - z_0| < R$.

16.5 Term-by-term integration

On any contour $C$ inside the disk of convergence: $$\int_C f(z)\, g(z)\, dz = \sum_{n=0}^\infty a_n \int_C g(z)(z - z_0)^n\, dz$$ (uniform convergence allows swapping integral and sum).

Module 17 — Taylor and Laurent Series ⭐

If $f$ is analytic, you can expand it as a power series. If $f$ has a singularity, you extend with negative powers — the Laurent series. The coefficient $b_1$ in the Laurent series will become THE residue.

17.1 Taylor's theorem

If $f$ is analytic on $|z - z_0| < R$, then for all $z$ in this disk: $$f(z) = \sum_{n=0}^\infty \frac{f^{(n)}(z_0)}{n!}(z - z_0)^n.$$
In real analysis, you need extra hypotheses (bounded derivatives) for Taylor series to converge to $f$. In complex, analytic alone is enough!
Proof idea
Use CIF. For $z$ inside $|s - z_0| = R'$, write $1/(s-z) = \sum (z-z_0)^n / (s-z_0)^{n+1}$ (geometric series). Plug into CIF and integrate term by term — the coefficients become $f^{(n)}(z_0)/n!$ by the extended CIF.

Common Taylor series at $z_0 = 0$

$\dfrac{1}{1-z} = \sum_{n=0}^\infty z^n$, valid for $|z|<1$
$e^z = \sum \dfrac{z^n}{n!}$, valid everywhere
$\sin z = \sum \dfrac{(-1)^n z^{2n+1}}{(2n+1)!}$, valid everywhere
$\cos z = \sum \dfrac{(-1)^n z^{2n}}{(2n)!}$, valid everywhere

17.2 Laurent series

If $f$ is analytic on an annulus $R_1 < |z - z_0| < R_2$, then for all $z$ in the annulus: $$f(z) = \underbrace{\sum_{n=0}^\infty a_n (z - z_0)^n}_{\text{regular part}} + \underbrace{\sum_{n=1}^\infty \frac{b_n}{(z - z_0)^n}}_{\text{principal part}}.$$ The coefficients are uniquely determined by $f$.

17.3 Practical: compute series by tricks, not the formula

The textbook formula for $a_n, b_n$ is rarely useful. Instead:

Laurent series of $f(z) = \dfrac{1}{z(1+z^2)}$ for $0 < |z| < 1$.
Solution
$\dfrac{1}{1+z^2} = \dfrac{1}{1-(-z^2)} = \sum_{n=0}^\infty (-z^2)^n = \sum_{n=0}^\infty (-1)^n z^{2n}$ for $|z| < 1$.
Multiply by $1/z$: $$f(z) = \frac{1}{z} - z + z^3 - z^5 + \cdots$$ The principal part is just $1/z$; coefficient $b_1 = 1$.
Same function $f$, but for $|z| > 1$.
Solution
Now $|1/z^2| < 1$. Write $\dfrac{1}{1+z^2} = \dfrac{1}{z^2}\dfrac{1}{1 + 1/z^2} = \dfrac{1}{z^2}\sum (-1/z^2)^n = \sum \dfrac{(-1)^n}{z^{2n+2}}$.
Multiply by $1/z$: $f(z) = \sum \dfrac{(-1)^n}{z^{2n+3}}$ for $|z| > 1$. Different region, different series!
Series depends on the annulus. Same function, same center, different annulus → different series. Always specify the region.

17.4 Identity theorem

If $f, g$ analytic on a domain $D$ and $f = g$ on a small subdomain (or even on a non-trivial line segment, or on any sequence with a limit point in $D$), then $f \equiv g$ on all of $D$.

Stunning consequence: tiny information determines the function globally. (Compare to real analysis where you can patch smoothly varying functions on disjoint pieces.)

Module 18 — Isolated Singularities

18.1 Three types

$z_0$ is an isolated singularity of $f$ if $f$ is analytic on some deleted neighborhood $0 < |z - z_0| < r$, but not at $z_0$.

Look at the principal part of the Laurent series at $z_0$:

TypePrincipal partEquivalent
Removable$= 0$ (no negative powers)$f$ is bounded near $z_0$. Define $f(z_0) = a_0$ and $f$ becomes analytic.
Pole of order $m$finitely many terms, largest is $b_m/(z-z_0)^m$$|f(z)| \to \infty$ as $z \to z_0$. Equivalently $(z-z_0)^m f(z)$ is analytic and nonzero at $z_0$.
Essentialinfinitely many nonzero $b_n$Casorati–Weierstrass: image of any deleted neighborhood is dense in $\mathbb{C}$ (function "goes everywhere").

18.2 Examples

$f(z) = \dfrac{\sin z}{z}$ at $z=0$: $\sin z = z - z^3/6 + \cdots$, so $f(z) = 1 - z^2/6 + \cdots$ — Taylor series, no principal part. Removable. Defining $f(0) = 1$ makes $f$ entire.
$f(z) = \dfrac{1}{z^2(1-z)}$ at $z=0$: expand $\dfrac{1}{1-z} = 1 + z + z^2 + \cdots$, so $f = \dfrac{1}{z^2} + \dfrac{1}{z} + 1 + z + \cdots$. Largest negative power is $1/z^2$. Pole of order 2.
$f(z) = e^{1/z}$ at $z=0$: $e^{1/z} = \sum \dfrac{1}{n!\, z^n}$. Infinitely many negative powers. Essential.

18.3 Zeros and poles

$f$ has a zero of order $m$ at $z_0$ if $f(z) = (z-z_0)^m g(z)$ where $g$ is analytic and $g(z_0) \neq 0$.
Zeros of a non-constant analytic function are isolated.
If $f_1(z_0) \neq 0$ and $f_2$ has a zero of order $m$ at $z_0$, then $f_1/f_2$ has a pole of order $m$ at $z_0$.

Module 19 — Residues ⭐⭐⭐

The residue is just the coefficient $b_1$ in the Laurent series. But it controls the entire integral around the singularity. This is the most powerful tool you'll learn.

19.1 Definition

If $z_0$ is an isolated singularity of $f$ with Laurent expansion $\sum a_n (z-z_0)^n + \sum b_n/(z-z_0)^n$, the residue is $$\operatorname{Res}_{z = z_0} f = b_1.$$

Equivalent integral definition: for any small circle $C$ around $z_0$,

$$\operatorname{Res}_{z=z_0} f = \frac{1}{2\pi i}\oint_C f(z)\,dz.$$

19.2 Cauchy's residue theorem

Let $C$ be a simple closed contour (counterclockwise) and $f$ analytic on and inside $C$ except at finitely many isolated singularities $z_1, \ldots, z_n$ inside $C$. Then $$\oint_C f(z)\,dz = 2\pi i \sum_{k=1}^n \operatorname{Res}_{z = z_k} f.$$

19.3 How to compute residues — practical formulas

Simple pole (order 1)

$$\operatorname{Res}_{z=z_0} f = \lim_{z \to z_0} (z - z_0) f(z).$$

Quotient at simple pole

If $f = p/q$, $p(z_0) \neq 0$, $q(z_0) = 0$ but $q'(z_0) \neq 0$: $$\operatorname{Res}_{z=z_0} \frac{p}{q} = \frac{p(z_0)}{q'(z_0)}.$$

Pole of order $m$

Let $\varphi(z) = (z - z_0)^m f(z)$ (which becomes analytic and nonzero at $z_0$). Then $$\operatorname{Res}_{z=z_0} f = \frac{\varphi^{(m-1)}(z_0)}{(m-1)!}.$$

19.4 Worked examples

$\operatorname{Res}_{z=i} \dfrac{z+4}{z^2+1}$.
Solution
$z^2+1 = (z-i)(z+i)$. Simple pole at $z=i$. Use $p/q'$ formula: $p(i) = i+4$, $q'(z) = 2z$, $q'(i) = 2i$. Residue $= (i+4)/(2i) = 1/2 - 2i$ (after simplification).
$\operatorname{Res}_{z=i} \dfrac{z^3 + 2z}{(z-i)^3}$.
Solution
Pole of order 3. $\varphi(z) = (z-i)^3 f(z) = z^3 + 2z$. Take $\varphi''(z)$ and divide by $2!$:
$\varphi'' = 6z$. So $\operatorname{Res} = 6i / 2 = 3i$.
$\operatorname{Res}_{z=0} \dfrac{1 - \cos z}{z^3}$ — careful, looks like order 3, but isn't!
Solution
$1 - \cos z = z^2/2 - z^4/24 + \cdots$. So $f(z) = (z^2/2 - z^4/24 + \cdots)/z^3 = 1/(2z) - z/24 + \cdots$.
The actual order is 1, and $\operatorname{Res} = 1/2$.
Lesson: always verify $\varphi(z_0) \neq 0$ before using the order-$m$ formula! Otherwise you got the order wrong.

19.5 Residue at infinity (occasionally useful)

For $f$ analytic outside some disk: $$\operatorname{Res}_{z=\infty} f = -\operatorname{Res}_{w=0}\frac{1}{w^2} f\!\left(\frac{1}{w}\right).$$

If $C$ encloses all singularities, $\oint_C f\, dz = -2\pi i\,\operatorname{Res}_{\infty} f$. Useful when there are many finite singularities — sometimes the residue at $\infty$ is easier.

Compute $\oint_{|z|=2} \dfrac{z+1}{z(z-1)}\,dz$.
Solution
Poles at $z=0$ (residue $= (0+1)/(0-1) = -1$) and $z=1$ (residue $= (1+1)/1 = 2$). Both inside.
Integral $= 2\pi i (-1 + 2) = 2\pi i$.

Module 20 — Real Integrals via Residues (Rational Functions)

Now we cash in. Hard real integrals — that you'd need clever substitutions or have no elementary form — fall to residues in two lines.

20.1 The recipe

To compute $\int_{-\infty}^\infty f(x)\,dx$ where $f$ is rational, no real poles, $\deg(\text{denom}) \geq \deg(\text{numer}) + 2$:

  1. Replace $x$ with complex $z$: get $f(z)$.
  2. Take the contour: $[-R, R]$ on the real axis, plus the upper semicircle $C_R$.
  3. By the residue theorem: $\int_{-R}^R + \int_{C_R} = 2\pi i \sum (\text{residues in upper half-plane})$.
  4. Show $\int_{C_R} \to 0$ as $R \to \infty$ via the ML inequality.
  5. Get $\int_{-\infty}^\infty f(x)\,dx = 2\pi i \sum (\text{residues in UHP})$.

20.2 Detailed example: $\int_0^\infty \dfrac{dx}{x^6 + 1}$

Full solution
Step 1. $f(z) = 1/(z^6+1)$. Singularities: $z^6 = -1$, so $z_k = e^{i(2k+1)\pi/6}$ for $k = 0, \ldots, 5$. Three are in the upper half plane: $z_0 = e^{i\pi/6}$, $z_1 = e^{i\pi/2} = i$, $z_2 = e^{i 5\pi/6}$.
Step 2. Each is a simple pole. Use $p/q'$ formula: $\operatorname{Res} = 1/(6 z_k^5)$.
Since $z_k^6 = -1$, $z_k^5 = -1/z_k$. So $\operatorname{Res} = -z_k/6$.
Step 3. Sum: $-(z_0 + z_1 + z_2)/6$.
$z_0 + z_2 = 2\cos(\pi/6) \cdot e^{i\pi/2} = i\sqrt 3$ (using sum-to-product, or just imag parts: $\sin(\pi/6) + \sin(5\pi/6) = 1$, real parts cancel).
Actually let me recompute: $z_0 = \cos(\pi/6) + i\sin(\pi/6) = \tfrac{\sqrt 3}{2} + \tfrac{i}{2}$. $z_2 = \cos(5\pi/6) + i\sin(5\pi/6) = -\tfrac{\sqrt 3}{2} + \tfrac{i}{2}$. Sum: $i$. Plus $z_1 = i$. Total $= 2i$.
So sum of residues $= -2i/6 = -i/3$.
Step 4. $\int_{-R}^R + \int_{C_R} = 2\pi i \cdot (-i/3) = 2\pi/3$.
Step 5. On $C_R$: $|f| \leq 1/(R^6 - 1)$, length $\pi R$. So $|\int_{C_R}| \leq \pi R/(R^6 - 1) \to 0$.
Step 6. $\text{PV}\int_{-\infty}^\infty \dfrac{dx}{x^6+1} = 2\pi/3$. Since integrand is even, $\int_0^\infty = \pi/3$. ✓

20.3 Why $\int_{C_R} \to 0$

For a rational function $f = P/Q$ with $\deg Q \geq \deg P + 2$, on $|z| = R$ large: $|f(z)| \leq M/R^2$ approximately. Length of $C_R$ is $\pi R$. So $|\int_{C_R}| \leq \pi R \cdot M/R^2 = \pi M/R \to 0$.

The degree gap of at least 2 is what makes this work. With gap of just 1, you'd need Jordan's lemma (next module).

Module 21 — Real Integrals with $\sin/\cos$ & Jordan's Lemma

21.1 The trick: replace $\cos ax, \sin ax$ with $e^{iaz}$

Don't use $\cos z$ or $\sin z$ in your contour integral! They're unbounded on the upper half plane.
Instead, use $e^{iaz}$: on the upper half plane ($y \geq 0$, $a > 0$), $$|e^{iaz}| = |e^{ia(x+iy)}| = e^{-ay} \leq 1.$$ Bounded! Then take real or imaginary part of the answer.

21.2 Worked example

Compute $\int_0^\infty \dfrac{\cos 2x}{(x^2 + 4)^2}\, dx$.
Full solution
Step 1. $f(z) = \dfrac{e^{i 2z}}{(z^2+4)^2}$. The desired integral is $\tfrac{1}{2}\operatorname{Re}\,\text{PV}\int_{-\infty}^\infty f(x)\,dx$ (using evenness).
Step 2. Singularities: $z = \pm 2i$. Only $z = 2i$ in UHP. It's a pole of order 2.
Step 3. $\varphi(z) = (z-2i)^2 f(z) = e^{i2z}/(z+2i)^2$. Take derivative: $$\varphi'(z) = \frac{2i(z+2i)^2 e^{i2z} - e^{i2z}\cdot 2(z+2i)}{(z+2i)^4} = e^{i2z}\frac{2i(z+2i) - 2}{(z+2i)^3}.$$ At $z = 2i$: $\varphi'(2i) = e^{-4}\cdot \dfrac{2i(4i) - 2}{(4i)^3} = e^{-4}\cdot\dfrac{-8 - 2}{-64i} = \dfrac{-10 e^{-4}}{-64 i} = \dfrac{10 e^{-4}}{64 i} = \dfrac{5 e^{-4}}{32 i}$.
$\operatorname{Res} = \varphi'(2i) = \dfrac{5 e^{-4}}{32 i} = -\dfrac{5i e^{-4}}{32}$.
Step 4. $\int_{-R}^R + \int_{C_R} = 2\pi i \cdot (-5i e^{-4}/32) = \dfrac{10\pi e^{-4}}{32} = \dfrac{5\pi e^{-4}}{16}$.
Step 5. On $C_R$: $|f| \leq 1 \cdot 1/(R^2-4)^2$. Times length $\pi R$: $\to 0$.
Step 6. $\text{PV}\int_{-\infty}^\infty f(x)\,dx = 5\pi e^{-4}/16$. Take real part (it's already real): $5\pi e^{-4}/16$.
By evenness: $\int_0^\infty \cos 2x/(x^2+4)^2\, dx = 5\pi/(32 e^4)$.

21.3 Jordan's lemma

When the degree gap isn't enough for the basic ML estimate, but you have an $e^{iaz}$ factor:

Suppose $g(z)$ is analytic on the upper half plane outside some disk, and $\max_{C_R} |g(z)| = M_R \to 0$ as $R \to \infty$. Then for any $a > 0$: $$\int_{C_R} g(z)\, e^{iaz}\, dz \to 0.$$
Proof using a clever bound
$|e^{iaz}| = e^{-ay} = e^{-aR\sin\theta}$ on $C_R$. Using Jordan's inequality $\sin\theta \geq 2\theta/\pi$ for $\theta \in [0, \pi/2]$: $$\int_{C_R} |g\, e^{iaz}|\,|dz| \leq M_R\, R \int_0^\pi e^{-aR\sin\theta}\,d\theta \leq M_R \cdot R \cdot \frac{\pi}{aR} = \frac{\pi M_R}{a} \to 0.$$

21.4 Example needing Jordan

$\int_0^\infty \dfrac{x \sin 2x}{x^2 + 3}\, dx = \dfrac{\pi}{2} e^{-2\sqrt 3}$.
Why we need Jordan
$f(z) = z e^{i 2z}/(z^2 + 3)$. The naive ML bound gives $|f| \leq R/(R^2 - 3)$, length $\pi R$, so bound $\sim \pi R^2/R^2 \not\to 0$.
But write $f(z) = g(z) e^{i 2z}$ with $g(z) = z/(z^2+3)$. On $C_R$: $|g| \leq R/(R^2 - 3) \to 0$. So Jordan's lemma applies. ✓

Module 22 — Indented Paths & Branch Cut Contours

22.1 When there's a pole on the real axis

If $f$ has a simple pole at a real point $x_0$, you can't run your contour through it. Indent with a small semicircle.

If $f$ has a simple pole at $z = x_0 \in \mathbb{R}$ with residue $r$, and $C_\rho$ is the small clockwise upper semicircle of radius $\rho$ around $x_0$, then $$\lim_{\rho \to 0^+} \int_{C_\rho} f(z)\,dz = -i\pi\, r.$$

Half of $2\pi i$, with a sign flip because we go clockwise. (If you go counterclockwise instead, it's $+i\pi r$.)

22.2 The classic: Dirichlet integral

$\displaystyle\int_0^\infty \frac{\sin x}{x}\, dx = \frac{\pi}{2}$.
Full solution
Step 1. $f(z) = e^{iz}/z$. Singularity at $z = 0$ — on the real axis!
Step 2. Use the indented contour: $[-R, -\rho]$, small semicircle $C_\rho$ around 0 (clockwise), $[\rho, R]$, large semicircle $C_R$.
Step 3. No singularities inside, so the closed-contour integral is $0$.
Step 4. By Jordan (applied to $g(z) = 1/z$, $|g| = 1/R$ on $C_R$): $\int_{C_R} \to 0$.
Step 5. Indented-pole result: $\operatorname{Res}_{z=0} (e^{iz}/z) = 1$. So $\int_{C_\rho} \to -i\pi$.
Step 6. $\int_{-R}^{-\rho} + \int_{\rho}^R + (\to 0) + (\to -i\pi) = 0$. So
$\text{PV}\int_{-\infty}^\infty \dfrac{e^{ix}}{x}\,dx = i\pi$.
Take imaginary part: $\text{PV}\int \dfrac{\sin x}{x}\,dx = \pi$. By evenness (of $\sin x/x$): $\int_0^\infty = \pi/2$. ✓

22.3 Branch cut contours

For integrals like $\int_0^\infty x^{-a}/(x+1)\,dx$, the branch cut is along the positive real axis. Use a "keyhole" contour:

  • Line just above positive real axis (outgoing): $z = x$, gives $x^{-a}$.
  • Big circle $C_R$.
  • Line just below positive real axis (returning): $z = xe^{2\pi i}$, gives $x^{-a} e^{-2\pi i a}$.
  • Small circle $C_\rho$ around 0.
$\displaystyle\int_0^\infty \frac{x^{-a}}{x + 1}\, dx = \frac{\pi}{\sin a\pi}$ for $0 < a < 1$.
Full solution
$f(z) = z^{-a}/(z+1)$ with branch $\theta \in (0, 2\pi)$.
Above real axis: $z = x$, integrate $\int_\rho^R f(x)\,dx$.
Below real axis: $z = xe^{2\pi i}$, $z^{-a} = x^{-a} e^{-2\pi i a}$. Going right-to-left is opposite orientation. Integral becomes $-e^{-2\pi i a}\int_\rho^R f(x)\,dx$.
Big circle: $|f| \leq R^{-a}/(R-1)$, length $2\pi R$. Bound $\sim R^{1-a}/R \to 0$ since $a > 0$.
Small circle: $|f| \leq \rho^{-a}/(1-\rho)$, length $2\pi\rho$. Bound $\sim \rho^{1-a} \to 0$ since $a < 1$.
Single pole inside: $z = -1$, residue $= (-1)^{-a} = e^{-i\pi a}$ (using branch).
Combine: $(1 - e^{-2\pi i a})\int_0^\infty f(x)\,dx = 2\pi i \cdot e^{-i\pi a}$.
Multiply numerator and denominator by $e^{i\pi a}$: $$\int_0^\infty \frac{x^{-a}}{x+1}\,dx = \frac{2\pi i\, e^{-i\pi a}}{1 - e^{-2\pi i a}} = \frac{2\pi i}{e^{i\pi a} - e^{-i\pi a}} = \frac{\pi}{\sin a\pi}. \quad \blacksquare$$

  • Hard real integrals become easy via residues.
